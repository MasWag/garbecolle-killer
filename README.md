garbecolle-killer
=====================

bot for https://github.com/m-hys/garbecolle

Requisite
===========

* g++ >= 4.8
* opencv2
* omake
* xautomation

How to run
=============

1. prepare garbecolle
2. open terminal and run garbecolle-killer
3. switch display to garbecolle(you have only 5 secs)
4. nerds will be collected
